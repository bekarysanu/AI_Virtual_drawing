#AI Virtual Drawing Board

An interactive computer vision application that enables users to draw in the air using hand gestures detected through a webcam.
The system uses MediaPipe hand tracking and OpenCV to interpret finger gestures as drawing commands and illustrate them in real time in a virtual canvas.

This project demonstrates the implementation of gesture recognition, real-time image processing, and human and computer cooperation using computer vision.

Also it was developed to explore practical applications if computer vision and gesture-based interface.
It demonstrates how real-time hand tracking can be used to build intuitive computer interaction system.

## Features 
-Hand gesture drawing
-Color selection panel
-Gesture-based eraser
-Dynamic brush size control
-Undo gesture
-Save the drawing 
-Screenshot mode
-Real-time FPS counter
-Fullscreen Interface

## Technologies
-Python
-OpenCV
-MediaPipe
-NumPy
